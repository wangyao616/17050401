|   名称   |                  值                   |
| :------: | :-----------------------------------: |

|   语言   |                 Java                  |

#### 功能描述： 

- 打开应用程序，直接显示原题答案
- 之后功能强化了一下，具体是：将**年份**、     字长位数、一名程序员每天可开发出的指令数和程序员的平均月薪参数化了，可以根据实际情况输入这些参数，从而得到相应的结果
- 当输入的参数不符合要求的时候：相应的文本框会有提示信息显示
